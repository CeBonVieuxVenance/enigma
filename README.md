Ce projet à pour objectif de recréer la machine enigma. Pour ce qui est de la partie historique je vous laisse vous renseigner sur Wikipédia (il vous renseignera beaucoup mieux que moi). 
L'objectif ici est vraiment de se renseigner sur l'aspect technique de la machine qui est composée des parties suivantes :

1 - le clavier 

2- les branchements 

3- les rotors 

Bien évidemment cela est vrai pour une machine Enigma électro-mécanique dands le cas du code, le clavier est remplacé par un texte directement donné à l'éxecution.
